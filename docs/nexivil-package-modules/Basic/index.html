    Basic

    Class#

    Point#

    Coordinate with information on the local coordinate system from which this is derived.

    Constructor#

    Point(x, y, z)

      x Number : value of x-axis [mm]

      y Number : value of y-axis [mm]

      z Number : value of z-axis [mm]

    Properties#

    Containing the data of constructor input.

    RefPoint#

    Coordinate with plane information.

    Constructor#

    RefPoint(originPoint = new Point(0,0,0), xAxis = new Point(1,0,0), xRotation = PI/2)

      originPoint Point : Point() with coordinate values

      xAxis Point : Point() with direction vectors of x-axis

      xRotation Number : x-axis rotation angle to be equal to the global coordinate system [rad]

    Properties#

    See the base Point class for inherited properties.

      .xAxis Point : Point() with x-axis direction vectors of UCS in global coordinate system

      .xRotation Number : x-axis rotation angle to be equal to the global coordinate system [rad]

      .yRotation Number : y-axis rotation angle to be equal to the global coordinate system [rad]

      .zRotation Number : z-axis rotation angle to be equal to the global coordinate system [rad]

    Alignment#

    Infromation of alignment with points.

    Constructor#

    Alignment(horizonData, verticalData, superElevation, beginStation)

      horizonData Double array : Basic input to create alignment ( element : [ x, y, r, a1, a2 ] )

        x Number : coordinate value of x-axis [mm]

        y Number : coordinate value of y-axis [mm]

        r Number : radius of clothois [mm]

        a1 Number : constant for calculate starting clothoid

        a2 Number : constant for calculate end clothoid

      verticalData Array : Vertical information of alignment ( Element: [ station, elevation, curveLength ] )

        station Number : Distance from the begin station to a specific station [mm]

        elevation Number : Height on the station position

        curveLength Number : ...

      superElevation Array: Elevation information of alignment ( Element: [ station, left, right ] )

        station Number : Distance from the begin station to a specific station [mm]

        left Number : ...

        right Number : ...

    Properties#

      .HorizonDataList Array : horizonData of input

      .VerticalDataList Array : verticalData of input

      .SuperElevation Array : superElevation of input

      .beginStation Number : beginStaion of input

      .vectors Array : vector info created with coordinates of horizonData

      .curves Array : clothoid info

      .segments Array : Array with element contained start and end station of each segments for clothoid

      .startPoint Array : ... (array 형태? 무엇을 의미?)

      .tanget Array : ... (array 형태? 무엇을 의미?)

      .parabolaData Array : ...

      .endStation Number : Station value of end point

      .points Array : Array with element contained super elevation, vertical info, normal vector, skew info

    StPoint#

    Coordinate with alignment information.

    Constructor#

    StPoint(station, line, skew = PI/2, offset = 0)

      station Number : Distance from the starting point of the alignment to this point

      line Alignment or Spline : ..

      skew Number : Skew (default = PI/2) [rad]

      offset Number : Distance in normal direction

    Properties#

    Contains the data of constructor input.

    See the base RefPoint class for inherited properties.

      .station number : Distance from the begin station of the alignment to this point

      .mainStation number : Distance from the init point of the alignment to this point (station of inputs)

      .gradientX number : Longitutinal gradient (x-axis rotation angle) [rad]

      .gradientY number : Horizontal gradient (y-axis rotation angle) [rad]

      .leftGradient number : Superelevation to the left [rad]

      .rightGradient number : Superelevation to the right [rad]

    Spline#

    A 3D curve of array type with more than 2 points.

    Constructor#

    Spline(refPoints, alignment = null)

      refPoints Array : A Array with RefPoint() as an element

      alignment Alignment : ..

    Properties#

      .points Array : RefPoints with station

    Module#

    PointLengthXY#

    Calculate the length from the origin to this point projected on th xy plane.

    PointLengthXY(point)

    Inputs#

      point Point

    Output#

      Number

    PointLength#

    Calculate the length from the origin to this point.

    PointLength(point)

    Inputs#

      point Point

    Output#

      Number

    PointToLocal#

    Rotate the coordinate of point to refPoint with plane information and translate. (rotation order: Z -> Y -> X)

    PointToLocal(point, refPoint)

    Inputs#

      point Point or [Point...]

      refPoint RefPoint

    Output#

      Point or [Point...]

    PointToGlobal#

    Translate the coordinate of point to refPoint with plane information and rotate. (rotation order: -X -> -Y -> -Z)

    PointToGlobal(point, refPoint)

    Inputs#

      point Point or [Point...]

      frePoint RefPoint

    Output#

      Point or [Point...]

    PointToSkewed#

    After transfrom point as skew, translate and rotate the translated point to stPoint (xRotation contained in stPoint is always PI/2) result = (transformed point - stPoint) * rotation

    PointToSkewed(point, stPoint)

    Inputs#

      point Point or [Point...]

      stPoint StPoint

    Output#

      Point or [Point...]

    StPointToParallel#

    A point on the alignment that is away from stPoint by station value and has a normal vector equal to the normal vector of the stPoint plane.

    StPointToParallel(stPoint, offset, alignment)

    Inputs#

      stPoint StPoint

      offset Number : Distance from the stPoint to a location to define on the alignment

      alignment Alignment

    Output#

      StPoint

    AlignmentToOffsetSpline#

    Translate alignment by offset in the normal direction of each alignment's point.

    AlignmentToOffsetSpline(alignment, offset, startPoint, endPoint)

    Inputs#

      alignment Alignment

      offset Number : Distance from the each points on the alignment

      startPoint StPoint : External alignment point corresponding to start position of spline (default = null)

      endPoint StPoint : External alignment point corresponding to end position of spline (default = null)

    Output#

      Spline

    MatchedPointOnAlignment#

    Find the intersection StPoint between the alignment and a plane contatined the point with normal vector parallelt to the x-axis.

    MatchedPointOnAlignment(alignment, point)

    Inputs#

      alignment Alignment

      point Point

    Output#

      StPoint

    IntersectionPointOnSpline#

    Find the intersection RefPoint between the spline and a plane contained the stPoint.

    IntersectionPointOnSpline(spline, stPoint, alignment)

    Inputs#

      spline Spline

      stPoint StPoint

      alignment Alignment : default = null

    Output#

      StPoint
