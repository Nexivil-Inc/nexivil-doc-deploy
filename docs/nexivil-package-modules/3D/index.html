    Model#

    Create a new model class with meta information of user.

    Constructor#

    Model(part = "default", key = "default", material = "default")

      part String : The title of model (default = "default")

      key String : The subtitle of model (default = "default")

      material String : The material name of model (default = "default")

    Properties#

      .meta Object : {part, key, material}

    Loft#

    Create a new Loft class with meta information of user.

    Constructor#

    Loft(points, closed = true, material = "default", part = "default", key = "default")

      points Array : The points for section

      closed Boolean : Wheter to connect the start and end in points

      material String : The material name of model (default = "default")

      part String : The title of model (default = "default")

      key String : The subtitle of model (default = "default")

    Properties#

      .type String : "loft"

      .points Array

      .closed Boolean

      .cap String : Equal to .closed

      .meta Object : {part, key, material}
