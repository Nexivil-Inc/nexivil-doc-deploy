패키지 업데이트

모델은 템플릿 패키지에 기반하여 생성됩니다. 템플릿 패키지가 업데이트되고 그 업데이트가 모델에 반영될려면 작업이 필요합니다.

업데이트 방법#

  • 버튼 클릭
  • Check Update 바 확장 후 스위치버튼 토글 그리고 CONFIRM 버튼 클릭
  • Update Detected 바 확장 후 UPDATE 버튼 클릭
  • Require Restart 바 확장 후 RESTART 버튼 클릭

OR

  • 업데이트 대상 없을시 Up To Date 바

info
버튼이 있던 위치의 모양이 바뀔때마다 버튼이 기능이 달라집니다. 달라지는 순서는 상기의 순서와 같습니다.
