1.선형정보

선형정보>주선형#

평면선형#

  • “x”: 평면도상의 선형 기준점(IP)의 경도 (상대)좌표(단위: mm)
  • “y”: 평면도상의 선형 기준점(IP)의 위도 (상대)좌표(단위: mm)
  • “r”: 곡선부 곡률반경, 시작점(BP), 끝점(EP)에서는 반영되지 않음(단위: mm)
  • “a1”: 곡선부 시작구간의 클로소이드 매개변수( = √(L*r), L: 클로소이드 곡선길이, 단위:mm)
  • “a2”: 곡선부 종료구간의 클로소이드 매개변수( = √(L*r), L: 클로소이드 곡선길이, 단위:mm)

시작스테이션#

  • 시작점(BP: 평면성형의 첫번째 행 기준점)의 스테이션 값(단위: mm)

편경사#

  • “station”: 편경사 기준점의 선형 스테이션(단위: mm)
  • “left”:선형진행방향기준 좌측부의 편경사(선형중심 기준으로 하향일 경우 음수, 상향일 경우 양수)
  • “right”:선형진행방향기준 우측부의 편경사(선형중심 기준으로 하향일 경우 음수, 상향일 경우 양수)

종단계획고#

  • “station”: 종단계획고 기준점의 선형 스테이션(단위: mm)
  • “elevation”: 선형 스테이션 좌표의 계획고(도로포장면 고도, 단위: mm)
  • “curveLength”: 계획고의 2차 포물선 적용길이(첫행과 마지막행 값은 반영되지 않음, 단위: mm)

선형정보>보조선형#

평면선형#

  • “x”: 평면도상의 선형 기준점(IP)의 경도 (상대)좌표(단위: mm)
  • “y”: 평면도상의 선형 기준점(IP)의 위도 (상대)좌표(단위: mm)
  • “r”: 곡선부 곡률반경, 시작점(BP), 끝점(EP)에서는 반영되지 않음(단위: mm)
  • “a1”: 곡선부 시작구간의 클로소이드 매개변수( = √(L*r), L: 클로소이드 곡선길이, 단위:mm)
  • “a2”: 곡선부 종료구간의 클로소이드 매개변수( = √(L*r), L: 클로소이드 곡선길이, 단위:mm)

시작스테이션#

  • 시작점(BP: 평면성형의 첫번째 행 기준점)의 스테이션 값(단위: mm)

선형정보>거더배치#

지점정보#

  • “angle”: 해당지점의 사각(직교일 경우: 90, 단위: DEG)
  • “spanLength”: 첫번째행은 교량의 시점으로 항상 0이 되어야 함. 두번째행부터 앞의 행과의 거리로 두번째행은 교량시점과 첫번째 받침점과의 선형거리, 마지막 행은 마지막 받침점과 - 교량종점과의 거리(단위: mm)

교량시점스테이션 : 선형상의 교량시작점 스테이션 값(단위: mm)#

거더배치#

  • 각행번호는 거더의 번호, 반드시 번호순으로 좌측에서 우측으로 거더 입력 요망
  • “baseLine”: 거더와 평행한 기준선형(주선형: MasterLine, 보조선형: Slave1)
  • “alignOffset”: 기준선형으로부터의 옵셋거리(음수는 좌측거리, 양수는 우측거리, 단위: mm)
  • “isBeam” : 외측보강보(I형강)일 경우 true, 일반 강상형 거더의 경우 false
