7.해석모델

해석모델>차선정보#

  • "거더번호": 추가하려는 차선과 가장 근접한 거더번호
  • "편심거리": 해당거도와 차선과의 편심거리(좌측의 경우 음수)

해석모델>포장하중#

  • "바닥판좌측기준": 포장하중의 시작점기준이 좌측일 경우 [V], 우측일 경우 [ ]}
  • "기준점거리- " 포장하중 시작점의 시작기준점 상대위치
  • "바닥판좌측기준": 포장하중의 끝점기준이 좌측일 경우 [V], 우측일 경우 [ ]}
  • "기준점거리- " 포장하중 끝점의 시작기준점 상대위치
  • "포장두께": 포장두께

해석모델>방호벽하중#

  • "바닥판좌측기준": 포장하중의 시작점기준이 좌측일 경우 [V], 우측일 경우 [ ]}
  • "기준점거리- " 방호벽하중의 시작기준점 상대위치
  • "단면적": 콘크리트 방호벽의 단면적

해석모델>차량정보#

  • "표준트럭": 도로교설계기준 준용
  • "차로하중": 도로교설계기준 준용
  • "피로트럭": 도로교설계기준 준용

해석모델>시공단계(입력 후 Apply 클릭)#

  • "기호- " - "l- ", 소문자 엘, loadCase의 첫문자
  • "l1": 단면: 비합성, 하중: 강상형 자중, 하부콘크리트 자중
  • "l2- "- " 단면: 1차합성(하부콘크리트 경화), 하중: 바닥판콘크리트 자중
  • "l3": 단면: 2차합성(장기, 3n), 하중: 포장하중(보도부), 방호벽(난간/방음벽)하중
  • "l4": 단면: 2차합성(단기, n), 하중: 지점침하, 풍하중(예정)
  • "l5": 단면: 2차합성(단기, n) 하중: 활하중(트럭하중, 차선하중, 피로트럭, 보도하중)
