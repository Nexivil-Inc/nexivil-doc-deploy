5.바닥판철근정보

바닥판철근정보>바닥판철근#

피복정보(철근 도심까지의 거리)#

  • "top": 바닥판 상면 피복두께
  • "side": 바닥판 측면 피복두께
  • "bottom": 바닥판 하면 피복두께
  • "blockOutH": 바닥판 단부 블록아웃 높이
  • "blockOutL- " : 바닥판 단부 블록아웃 길이

주철근정보(교축직각 방향 철근)#

  • "endSpacing": 단부 주철근 간격(상하부 동일)
  • "endUpperRebarDia": 단부 상부 주철근 지름
  • "endLowerRebarDia": 단부 하부 주철근 지름
  • "centerSpacing": 중앙부 주철근 간격(상하부 동일)
  • "centerUpperRebarDia": 중앙부 상부 주철근 지름
  • "centerLowerRebarDia": 중앙부 하부 주철근 지름
  • "supportSpacing": 연속지점부 주철근 간격
  • "supportUpperRebarDia": 연속지점부 상부 주철근 간격
  • "supportLowerRebarDia- " 연속지점부 하부 주철근 간격

배력철근정보(교축방향 철근)#

  • "endSpacing": 단부 배력철근 간격(상하부 동일)
  • "endUpperRebarDia": 단부 상부 배력철근 지름
  • "endLowerRebarDia": 단부 하부 배력철근 지름
  • "centerSpacing": 중앙부 배력철근 간격(상하부 동일)
  • "centerUpperRebarDia": 중앙부 상부 배력철근 지름
  • "centerLowerRebarDia": 중앙부 하부 배력철근 지름
  • "supportSpacing": 연속지점부 배력철근 간격
  • "supportUpperRebarDia": 연속지점부 상부 배력철근 간격
  • "supportLowerRebarDia- " 연속지점부 하부 배력철근 간격

바닥판철근정보>방호벽변수#

방호벽배치#

  • "type": 방호벽/포장/보도부/연석 선택, 방호벽단면정보 참고
  • "from": 방호벽배치를 위한 바닥판 기준점, 보도부의 경우 경사방향
  • "offset(slop)": 방호벽이 배치되는 바닥판 기준점과의 상대거리, 보도부의 경우 경사도(단위 %)

방호벽단면정보(공통)#

  • "w": 방호벽 폭
  • "h": 방호벽 상단 높이
  • "points": 방호벽 기준좌표

방호벽철근>주철근정보#

  • "spacing": 루프철근의 간격
  • "loopRebarDia": 루프철근의 지름
  • "longiRebarDia": 종방향철근의 지름
  • "shearRebarDia": 전단철근의 지름

방호벽철근>피복정보#

  • "top": 상면 피복두께
  • "side": 측면 피복두께
  • "bottom": 하면 피복두께

바닥판철근정보>하부콘크리트철근#

주철근정보#

  • spacing: 루프철근의 간격
  • "loopRebarDia": 루프철근의 지름
  • "longiRebarDia": 종방향철근의 지름

피복정보#

  • "top": 상면 피복두께
  • "side": 측면 피복두께
  • "bottom": 하면 피복두께
