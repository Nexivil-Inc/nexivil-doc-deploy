    2.단면기본정보

    단면기본정보>공통#

    • isFlat: 강상형 상부플랜지가 수평일 경우, [V] true, 바닥판 편경사와 동일할 경우, [ ] false
    • B: 강상형의 좌/우 복부판 내측 거리(하부플랜지 기준, 단위: mm)
    • SlabLeft: 거더 최좌측 중심과 바닥판 최좌측 단부와의 거리(단위: mm)
    • SlabRight: 거더 최우측 중심과 바닥판 최우측 단부와의 거리(단위: mm)
    • PavementT: 포장두께(단위: mm)

    단면기본정보>단부단면#

    • Box: 단부 형상이 박스인 경우, [V] true, 플레이트인 경우, [ ] false
    • H : 단부 형고(단면 중심축 기준 하부플랜지 상단부터 상부플랜지 하단까지의 거리, 단위: mm) 단, 단부바닥판 두께 증가 및 단부절취로 인한 형고변화를 반영하지 않은 형고 기준값
    • UF: 플레이트 거더부의 상부플랜지 기준폭(단위: mm)
    • LF: 플레이트 거더부의 상부플랜지 기준폭(단위: mm)
    • SlabH: 단부 바닥판 두께(단위: mm)
    • SlabEndH: 단부 바닥판 최외측 두께(단위: mm)
    • CutL: 단부 절취 길이(반영하지 않을 경우: 0, 단위: mm)
    • CutH: 단부 절취 길이(반영하지 않을 경우: 0, 단위: mm)

    단면기본정보>연속부단면#

    • H 연속지점부 형고(단면 중심축 기준 하부플렌지 상단부터 상부플렌지 하단까지의 거리, 단위: mm)
    • UF: 박스부의 상부플랜지 기준폭(단위: mm)
    • LF: 박스부의 상부플랜지 기준폭(단위: mm)
    • SlabH: 중앙부(연속부 포함)의 바닥판 두께(단위: mm)
    • SlabEndH: 중앙부(연속부 포함)의 바닥판 최외측 두께(단위: mm)
    • LCONC: 지점부 하부 콘크리트 기준 두께(단위: mm)
    • HaunchH: 중앙부 바닥판 헌치 기준 높이(상부플랜지 기울기가 편경사와 다르고 헌치의 높이가 충분하지 않을경우 자동 계산된 헌치의 높이가 음수가 나올
    • HaunchW : 헌치돌출길이, 상부플랜지 외측과 헌치시작부 사이 거리 (단위: mm)

    단면기본정보>시점정보#

    • A: 교량 시점의 신축이름 길이(교량시점부와 바닥판 시작점의 거리, 단위: mm)
    • B: 바닥판 돌출 길이(바닥판 시작점과 강상형 거더 시작점의 거리, 단위: mm)
    • C: 단부 거더 길이(강상형 거더 시작점과 바닥판 두께 변화 시작점과의 거리, 단위: mm)
    • D: 바닥판 두께변화부 길이(단위: mm)

    단면기본정보>종점정보#

    • 시점정보와 동일

    단면기본정보>자동생성추가변수#

    • diaSpacing: 다이아프램 기준간격(초기값 5000, 단위:mm)
    • segMaxLength: 현장이음으로 인한 세그먼트 분할 기준길이(단위: mm)
