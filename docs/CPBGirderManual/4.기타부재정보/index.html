4.기타부재정보

기타부재정보>현장이음 부재정보#

현장이음1~4#

  • "webJointWidth": 복부이음판 가로폭

  • "webJointHeight": 복부이음판 세로높이

  • "webJointThickness": 복부이음판 두께

  • "flangeJointLength": 상/하면외측이음판 가로길이

  • "flangeJointWidth": 상/하면내측이음판 세로폭

  • "flangeJointThickness": 상/하면이음판 두께(내측/외측 동일)

  • "webJointThickness": 복부이음판 두께

  • "webJointWidth": 복부이음판 가로폭

  • "uflangeJointThickness": 상부플랜지 내/외측이음판 두께

  • "lflangeJointThickness": 하부플랜지 내/외측이음판 두께

  • "uflangeJointLength": 상부플랜지 내/외측이음판 길이(교축방향)

  • "lflangeJointLength": 하부플랜지 내/외측이음판 길이(교축방향)

  • "margin2":플랜지 내측이음판 복부판/종리브 이격거리

  • "uRibJointHeight": 상부 종리브 이음판 폭,

  • "uRibJointLength": 상부 종리브 이음판 길이

  • "uRibJointThickness": 상부 종리브 이음판 두께

  • "lRibJointHeight": 하부 종리브 이음판 폭

  • "lRibJointLength": 하부 종리브 이음판 길이

  • "lRibJointThickness": 하부 종리브 이음판 두께

  • "webBoltPitch": 복부이음판 볼트 가로간격

  • "webBoltGauge": 복부이음판 볼트 세로간격

  • "webBoltDia": 복부이음판 볼트 지름

  • "flangeBoltPitch": 플랜지이음판 볼트 가로간격

  • "flangeBoltGauge": 플랜지이음판 볼트 세로간격

  • "flangeBoltDia": 플랜지이음판 볼트 지름 --- 

기타부재정보>수평부재정보#

수평보강재배치#

  • "from": 시작기준점 마크
  • "to": 끝기준점 마크
  • "startOffset": 수평보강재 시작점, 시작기준점 상대위치
  • "endOffset": 수평보강재 끝점, 끝기준점 상대위치
  • "width": 수평보강재의 폭
  • "thickness- " : 수평보강재의 두께
  • "chamfer": 모따기 길이
  • "isTop": 옵셋의 기준 플랜지가 상부플랜지인 경우 [V] true, 하부플랜지인 경우 [ ] false
  • "offset": 플랜지와 수평보강재의 이격거리

수평브레이싱배치#

  • "from": 시작기준점 마크
  • "to": 끝기준점 마크
  • "leftToRight": 왼쪽에서 시작해서 오른쪽으로 가로지르는 경우 [V] true, 반대인 경우 [ ] false
  • "section": 수평보강재(연결판) 단면 참고
  • "plateLayout": 연결판 배치

기타부재정보>잭업배치#

  • "기준점": 잭업보강재가 위치하는 지점 마크 예) - "G1S1- "
  • "상대거리": 보강재위치와 기준점과의 상대거리, 다수인 경우 - ",- " 콤마로 구분
  • "길이": 잭업보강재의 길이
  • "높이": 잭업보강재의 높이(폭)
  • "두께": 잭업보강재의 두께
  • "모따기": 잭업보강재 상단의 모따기 길이(양측 동일)
  • "양측배치": 잭업보강재가 복부판 내외측에 위치하는 경우 [V], 내측에만 위치하는 경우[ ]

기타부재정보>스터드배치#

  • "시작기준점": 시작기준점 마크
  • "끝기준점": 끝기준점 마크
  • "시작옵셋": 스터드배치 시작점, 시작기준점 상대위치
  • "끝옵셋": 스터드배치 끝점, 끝기준점 상대위치
  • "간격": 교축방향 스터드 간격
  • "배치": 교축직각방향 스터드 간격, 좌측플랜지 끝단을 기준으로 배치간격 입력 다수의 경우 - ",- " 콤마로 구분

기타부재정보>받침배치#

  • "기준점- " 지점의 마크, 예) - "G1S1":
  • "받침종류: 양방향단/횡방향가동/종방향가동/고정단 중 선택
  • "옵셋": 거더중심축과 받침의 횡방향 거리
  • "솔플레이트폭": 해당 받침의 솔플레이트 가로(횡방향) 폭
  • "솔플레이트길이": 해당 받침의 솔플레이트 세로(종방향) 길이
  • "솔플레이트두께": 받침 중심점 기준 솔플레이트의 두께
