주요부재정보 >기타변수#

Entrance#

  • "add" 복부판 단부 출입구 유[V]/무[ ]
  • "b1" 복부판 단부 출입구 폭
  • "h1" 복부판 단부 출입구 높이
  • "d1" 복부판 단부 출입구 하단 높이(하부플랜지 상면과 출입구 하단과의 거리)
  • "r" 출입구 필렛 반지름

주요부재정보 >강상형단면#

  • 공통: 첫행은 거더의 시작점을 기준, 해당 행은 끝점의 위치정보를 내포함.
  • 마지막 행은 반드시 기준점: "end" 상대거리:0 값이 존재해야 함.
  • 거더선택: girder0: 1번거더 "G1", girder1: 2번거더 "G2"
  • "기준점": 거더번호와 지점(기호"S")번호를 표기, 예) 1번거더 2번지점, "G1S2"
  • "상대거리": 구간의 끝점에 대한 기준점 기준 스테이션 거리, 음수일 경우 반대방향
  • "마크" 그리드상의 표기: 거더번호 "G1" + 기호 + "행번호"
  • 복부판/상부/하부플랜지두께의 기준점은 공장이음부와 동일함.

  • "배치": 강상형 중심축 기준 종리브 중심 위치(음수인 경우 좌측, 양수는 우측)

상부종리브#

  • "기호": "TR"
  • "판두께": 구간의 강판의 두께
  • "높이": 종리브의 높이
  • "배치": 강상형 중심축 기준 종리브 중심 위치(음수인 경우 좌측, 양수는 우측)

복부판두께#

  • "기호": "WF"
  • "판두께": 구간의 강판의 두께

형고#

  • "기호": "H"
  • "시작형고": 구간의 시작점의 형고
  • "끝형고": 구간의 끝점의 형고
  • "곡선형상": "straight": 직선, "circle":원곡선, "parabola": 2차포물선

하부콘크리트#

  • "기호": "LC"
  • "시작두께": 구간의 시작점의 하부콘크리트 두께
  • "끝두께": 구간의 끝점의 하부콘크리트 두께

주요부재정보>주요부재배치>#

  • 공통: 첫행은 거더의 시작점을 기준, 해당 행은 끝점의 위치정보를 내포함.
  • 마지막 행은 반드시 기준점: "end" 상대거리:0 값이 존재해야 함.
  • 거더선택: girder0: 1번거더 "G1", girder1: 2번거더 "G2"
  • "기준점": 거더번호와 지점(기호"S")번호를 표기, 예) 1번거더 2번지점, "G1S2"
  • "상대거리": 구간의 끝점에 대한 기준점 기준 스테이션 거리, 음수일 경우 반대방향
  • "마크" 그리드상의 표기: 거더번호 "G1" + 기호 + "행번호"

슬래브배치#

  • "기준점": "CRK0" 바닥판시점단부 시작점, "CRK2" 바닥판시점단부 끝점, "CRK3" 바닥판중앙부 시작점, "CRK4" 바닥판중앙부 끝점, "CRK5" 바닥판종점단부 시작점, "CRK7" 바닥판종점단부 끝점
  • 여기서 "CRK1", "CRK6"는 강상형의 시작점과 끝점을 의미함
  • "슬래브두께": 바닥판 두께
  • "캔틸레버두께": 바닥판 외측면 두께
  • "좌측거리": 최좌측 거더의 중심과 바닥판 좌측면의 거리
  • "우측거리": 최우측 거더의 중심과 바닥판 우측면의 거리
