# Template Editor 소개

비주얼 스크립팅 시스템인 템플릿 에디터는 노드 기반 인터페이스를 사용하여 3D 모델들을 만드는 개념을 토대로 한 비주얼 스크립팅 시스템입니다. 이 시스템은 매우 유연하고 강력해서, 프로그래머가 없으면 힘든 개념 구현이나 틀을 만드는 작업을 디자이너 혼자서 해낼 수 있게 도와주는 시스템입니다.
