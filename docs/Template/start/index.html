    Quick Start

    지금까지 알아본 시퀀스와 노드 구성을 이용해 손쉽게 3D cube 생성하는 템플릿을 작성합니다.

    Sequence 생성#

    템플릿 노드들을 구성할 시퀀스를 먼저 생성합니다.

    Node 구성#

    디자인 익스프레스에서 제공하는 노드 이외에 외부 노드들을 가져오기 위해서 싱크 버튼을 클릭합니다.

    Material 생성#

    드래그 앤 드롭으로 비주얼 스크립트에 머터리얼을 노드처럼 만들어서 연결해 사용할 수 있습니다.
